Planning Poker est une application Python basée sur Tkinter qui permet aux équipes de voter sur la complexité des fonctionnalités d\textquotesingle{}un projet à l\textquotesingle{}aide de la méthode de Planning Poker.\hypertarget{md_README_autotoc_md2}{}\doxysection{Fonctionnalités}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item Configuration du nombre de joueurs.
\item Chargement du backlog des fonctionnalités à partir d\textquotesingle{}un fichier JSON.
\item Utilisation de différentes règles de vote (strict, moyenne).
\item Affichage des résultats dans un tableau.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysection{Prérequis}\label{md_README_autotoc_md3}
